    Что такое Juddy?Это партнерская программа по конвертации мобильного трафика по средствам технологий WapClickНа данный момент есть множество решений по большой "Тройки" и под несколько типов трафика.Подключено:- 6 ЛП для мегафон (адалт, плеер, файлы)- 8 ЛП для мтс (адалт, плеер, игры, эротика, файлы)- 5 ЛП для билайн (адалт, кино)Детали:- Множество ЛП работают на высоких стартовых ребиллах, и индивидуально можно обсудить увеличение - Выплаты каждую неделю или по запросу- Для владельцев своих адалт-сайтов, есть отключение преленда по запросу- Арбитражникам - так же есть интересное предложение (актуально 2недели от 25.08.2014)
